Help - Input Failure Rate

The format of numerical value of failure rate for each module is:
    value1, value2, value3, ..... valueN
where, value1 is the numerical failure rate of module A, value2 is the numerical failure rate of module B, and so on.

For example, there are 4 modules in the system, Ra = 1.5, Rb = 2.0, Rc = 2.5, Rd = 0.8,
then you should input :

1.5, 2.0, 2.5, 0.8
 
If it reports that the numerical value is invalid, please check that whether what you input are correct.
1. all data should be numerical
2. all input data should be nonnegative
 
Input the fixed time:
You can specify a fixed time for which you want to get the reliability of system.
