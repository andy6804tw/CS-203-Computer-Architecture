Computer Architecture (ECE568/668)       

  • Outline:

    • I. Introduction
    • II. Performance Analysis (Ch.1 +)
    • III. Processor Design: Pipelining (App. C, Ch.3)
    • IV. Memory Design: Memory Hierarchy, Cache Memory, Secondary Memory (App. B, Ch.2)
    • V. Storage Systems and Input/Output (App. D)
    • VI. Vector Processors/ GPUs (App. G, Ch. 4)
    • VII. Power-Aware Computing

  • Prerequisites: ECE 232 (Hardware Organization & Design) or an equivalent course.
  • Recommended Textbook: D.A. Patterson and J.L. Hennessy, Computer Architecture: A Quantitative Approach, 5th edition, Morgan-Kaufmann, 2011.
  • Recommended reading:
    • J. P. Shen and M. Lipasti, Modern Processor Design: Fundamentals of Superscalar Processors, McGraw-Hill, 2005.
    • M. J. Flynn, Computer Architecture: Pipelined and Parallel Processor Design, Jones and Bartlett, 1995.
    • V. P. Heuring and H. F. Jordan, Computer Systems Design and Architecture, Addison-Wesley, 1997.
    • K. Hwang, Advanced Computer Architecture: Parallelism, Scalability, Programmability, McGraw-Hill, 1993.
