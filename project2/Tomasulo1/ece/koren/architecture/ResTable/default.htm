Reservation Table Analyzers

Introduction:

Two software tools for analysis of pipeline reservation tables (with of one or two tasks running on the same pipeline) are available. Given a pipeline these analyzers will calculate the state diagram, the greedy cycle, the minimum average latency, the forbidden latency, the pipeline collision vector, and the throughput for the pipeline.

  • A Simple (one task only) Reservation Table Analyzer
  • An advanced (one or two tasks) Reservation Table Analyzer

Software Requirements: Netscape 4.06 or higher or Internet Explorer 4.0 or higher with Java Virtual Machine 1.1.5 installed.
